The following explains the directory layout for the project.

Minutes: contains the minutes of all meetings
Documentation: contains the Group 5 documentation
Genesis: contains the project

To run the project do the following:

Navigate to the Genesis folder

Run the preinstall file found in the PreInstall folder before running the project.
It will install wxpython and matplotlib for you, if you have not already done so

To run the proper project, navigate to the scripts folder and find Genesis2.py.
Run that file to open the project and be able to test Genesis2
